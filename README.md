## 🇨🇳 中文

### 什么是一人CEO？

一人CEO 是专为**独立创业者、自由职业者、Solo Founder** 打造的 Web 管理平台。

一个人 = 销售 + 交付 + 财务 + 运营。这个平台把所有环节整合在一个界面中，让你专注创造价值，而不是在工具之间切换。

### 核心模块

| 模块 | 说明 |
|------|------|
| **总览** | 关键指标（MRR、总收入、活跃客户、进行中任务）+ AI 今日重点 + 每日原则 + 每日协议 + 突围清单 |
| **任务** | 看板 / 泳道双视图，支持拖拽排序，按客户关联，优先级与截止日管理 |
| **线索** | 销售漏斗看板（新线索→已联系→提案中→赢单），一键转化为签约客户 |
| **客户** | 签约客户管理，订阅制 / 项目制统一计费，阶段款追踪，税务配置 |
| **收支** | 收入支出记录，订阅收入自动生成，税务计算（加税/含税），趋势图表，CSV 导出 |

### 特色

- **极简设计** — 追求实用，无多余装饰，专注于信息密度和操作效率
- **离线优先** — PWA 架构，断网可用，恢复后自动同步
- **云端同步** — Supabase 实时数据同步，跨设备数据一致
- **中英双语** — 界面完整支持中文和英文切换
- **深色模式** — 系统级深色/浅色模式适配
- **每日系统** — 原则知识库（19 条）+ 5 步每日协议 + 突围任务清单，帮助建立工作节奏
- **小工具** — 精力电池、日历、番茄钟、快速笔记等可配置小工具

### 使用方式

访问 [soloceo.github.io/solo-ceo](https://soloceo.github.io/solo-ceo/) 即可使用。

支持添加到主屏幕作为独立应用运行（PWA），体验接近原生 App。

## 🇺🇸 English

### What is Solo CEO?

Solo CEO is a web-based management platform built for **solo entrepreneurs, freelancers, and indie founders**.

One person = sales + delivery + finance + operations. This platform brings every part of your business into a single interface so you can focus on creating value.

### Core Modules

| Module | Description |
|--------|-------------|
| **Dashboard** | Key metrics (MRR, Revenue, Active Clients, Tasks) + AI daily focus + Daily principles + Daily protocol + Breakthrough checklist |
| **Tasks** | Kanban / Swimlane dual view, drag-and-drop, client association, priority & due date management |
| **Leads** | Sales funnel board (New → Contacted → Proposal → Won), one-click convert to signed client |
| **Clients** | Client management, subscription / project billing, milestone payment tracking, tax configuration |
| **Finance** | Income & expense tracking, auto-generated subscription income, tax calculations (exclusive/inclusive), charts, CSV export |

### Highlights

- **Minimalist design** — Focused on information density and efficiency, no unnecessary decorations
- **Offline-first** — PWA architecture, works without internet, auto-syncs on reconnect
- **Cloud sync** — Supabase real-time sync across all devices
- **Bilingual** — Full Chinese and English interface support
- **Dark mode** — System-level dark/light mode
- **Daily system** — Principle library (19 entries) + 5-step daily protocol + breakthrough tasks to build work rhythm
- **Widgets** — Energy battery, calendar, pomodoro timer, quick notes, and more configurable widgets

### How to Use

Visit [soloceo.github.io/solo-ceo](https://soloceo.github.io/solo-ceo/) to get started.

Can be added to your home screen as a standalone app (PWA) for a native-like experience.

## Tech Stack

- **Frontend**: React 19 + TypeScript + Vite + Tailwind CSS v4
- **State**: Zustand
- **Backend**: Supabase (PostgreSQL + Auth + Realtime)
- **Offline**: sql.js (in-browser SQLite) + Service Worker (PWA)
- **Animation**: Motion (Framer Motion)
- **Charts**: Recharts
